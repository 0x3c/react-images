# react-images
## 技术盏
* react
* redux
* redux-thunk
* react-redux
* react-router v4
* es6
* fetch
* localStorage
* less 
* rem,flex
## Usage
```
// clone到本地
git clone git@github.com:jiadahao/react-images.git
// 安装依赖
yarn
// run
yarn start
``` 
yarn 有缓存机制，速度快，超好用
## 项目简介 
使用 `create-react-app` 脚手架搭建项目，配置 `px2rem` , `less` 。使用n年前百度图片的API,只专注于前端开发。图片以瀑布流方式显示， `localStorage` 存储搜索历史记录。
